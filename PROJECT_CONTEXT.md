# PROJECT_CONTEXT.md

## Tech Stack

- **Frontend:** Next.js 15 App Router, TypeScript, Tailwind, statisk export till `web/out`.
- **Backend:** Firebase Cloud Functions v2, Node 22 (uppgraderat från 20 i april 2026), TypeScript, region `europe-west1`.
- **Hosting:** Firebase Hosting.
- **Databas:** Firestore i `eur3`.
- **Auth:** Firebase Auth med Google som provider.
- **LLM:** OpenAI Responses API med modell via `OPENAI_MODEL`.
